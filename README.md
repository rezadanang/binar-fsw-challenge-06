# Car Management API

## Getting Started

1. Setting db postgres
2. Instal dependencies using yarn
```
yarn install
``
3. Create db postgres
```
sequelize db:create
```
4. Migrate db postgres
```
sequelize db:migrate
```
5. Seed db postgres
```
sequelize db:seed:all
```

6. Start project
```
yarn start
```
